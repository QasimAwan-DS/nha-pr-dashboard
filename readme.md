# NHA PR Command Center

**Interactive Road Safety Dashboard** for the **National Highway Authority (Pakistan)** — built as a technical + communication portfolio piece for the **Director Public Relations** position.

![NHA Dashboard](https://via.placeholder.com/800x400?text=NHA+PR+Dashboard) <!-- Replace with actual screenshot later -->

## 🎯 Purpose

This dashboard demonstrates how **data-driven insights** can support effective Public Relations, crisis communication, and stakeholder engagement for NHA.

It showcases my ability to blend **communication strategy** with **technical implementation** — exactly what a Director PR role requires.

## ✨ Key Features

- **Three Interactive Tabs**: Overview, Public Engagement, Crisis & Alerts
- **Realistic NHA Data**: Incidents, complaints, press releases, risk heatmap
- **Professional Export**: Generate official PDF reports with one click
- **Responsive Design**: Works perfectly on desktop and mobile
- **Dark/Light Mode** with Pakistan flag-inspired green theme
- **Live Refresh Simulation**

## 🛠 Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- Chart.js (Line, Bar, Pie charts)
- jsPDF + html2canvas (PDF generation)
- Lucide Icons
